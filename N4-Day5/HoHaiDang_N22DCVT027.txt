Hướng dẫn commit:
1.git int: khởi tạo Local Repository

2.git status: kiểm tra trạng thái các thư mục, tệp
tin trong Repository

3.git add . : add tất cả file vào trong vùng Staging Area.
  git add <file> : add 1 file vào trong vùng Staging Area.

4.git commit -m "new_vesion 1.0" để đưa các file ở vùng Staging Area lên Local Repository.
